# Security Policy & Privacy Commitment

## 1. Zero Data Collection (Cam kết quyền riêng tư)
I, **Tan (Van Tan Phan)**, firmly commit that this tool **does not collect, store, or transmit** any user data, personal information, or project files. 
* The script operates entirely on your local machine.
* No background processes or internet connections are used to "phone home."
* Your privacy and professional work are 100% safe.

## 3. Community Support & Integrity
This project is created to support the 3D community. The code is transparent and open-source so that anyone can inspect its safety. I am committed to maintaining a clean, helpful, and malicious-free environment for all artists.

* **Email:** Tanphan.Lct@gmail.com
